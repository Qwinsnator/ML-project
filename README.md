# Project Overview

## Files

- **Training_final.py**: contains the training pipeline for the machine learning models
- **Test_final.py**: used for evaluating the trained models on the test dataset
- **data_split.py**: splits the dataset into training and testing sets
- **ROC_curve_graph.py**: generates ROC curves to evaluate model performance


## Folders

- **Data/** :contains cleaned datasets used for training and testing
- **Results/** :stores model outputs and evaluation metrics
- **Visualization/** :contains plots and visual analysis of results
